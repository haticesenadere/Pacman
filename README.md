# Pac-Man

HTML5 Canvas ve saf JavaScript ile yazılmış klasik Pac-Man oyunu.

## Çalıştırma

`index.html` dosyasını tarayıcıda aç.

## Kontroller

- **Ok tuşları** veya **W / A / S / D**: hareket

## Dosyalar

- `index.html` — sayfa ve canvas
- `game.js` — oyun döngüsü, harita, duvar ve skor çizimi
- `pacman.js` — Pacman sınıfı (hareket, animasyon, yeme)
- `ghost.js` — Hayalet sınıfı (BFS ile yol bulma)
- `assets/` — görseller ve font
